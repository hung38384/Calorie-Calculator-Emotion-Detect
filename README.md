# Calorie-Calculator — UI & Privacy Notes

This project helps parents estimate calories and track digestive-friendly food choices for children.

Short UI/UX notes (Vietnamese):
- Giao diện đã được chuyển sang tiếng Việt cho các thành phần chính.
- Thêm chức năng lưu lịch sử bữa ăn (lưu localStorage). Các bữa ăn được lưu cục bộ trên trình duyệt.
- Thêm bảng Theo dõi tăng trưởng để nhập cân nặng/chiều cao và nhận đề xuất calo.
- Dữ liệu nhạy cảm như hình ảnh và lịch sử chỉ được lưu cục bộ; không gửi nơi công cộng trừ khi phụ huynh chủ động gửi cho chuyên gia.

If you want a production-ready privacy page or server-side storage, we can add encrypted storage and a consent flow.
